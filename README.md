Installation:
```
git clone https://github.com/omid-1985/nautilus-resize-images.git \
&& chmod +x nautilus-resize-images/install-resize-images.sh \
&& sh nautilus-resize-images/install-resize-images.sh \
&& rm -rf nautilus-resize-images/
```
